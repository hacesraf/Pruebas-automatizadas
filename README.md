

## Descripción

Proyecto de automatización de pruebas frontend utilizando Selenium IDE sobre una aplicación web pública 
y de pruebas de backend utilizando PostMan y Newman sobre un API publico.

Aplicación probada:
* https://www.saucedemo.com
* https://jsonplaceholder.typicode.com
---

# Herramientas utilizadas

* Newman
* PostMan
* Selenium IDE
* Selenium Side Runner


# Ejecución de pruebas

Ejecutar el siguiente comando en terminal:

selenium-side-runner Test_Frontend.side

---

# Resultado esperado

Las pruebas mostrarán resultados PASS/FAIL en consola dependiendo del resultado de cada caso de prueba.
